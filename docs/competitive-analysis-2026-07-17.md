**Date:** 2026-07-17
**Status:** Live analysis via real-time browser sweep
**Goal:** #1 in Georgia → #1 globally

## Executive Summary

sivrce888 has the strongest technical foundation of any Georgian real estate platform — 9-language i18n, PWA, multi-role dashboards, reviews/chat, and programmatic SEO that exceeds all competitors. However, three competitors have shipped AI features (SS.ge chat AI, Livo.ge Livo AI) and two have unique engagement mechanisms (SS.ge Stories, Korter.ge WhatsApp alerts) that we lack.

**Priority one: ship AI search.** It's the only feature that two competitors have and we have 0% of. Everything else is table stakes.

## Competitor Profiles

### 1. MyHome.ge — The Incumbent (#1 Claim)

- **Owner:** TNET group (Georgia's largest internet holding)
- **Auth:** auth.tnet.ge (shared with Livo)
- **Image CDN:** static-statements.tnet.ge
- **Key features:** Mortgage calculator, contract templates, price statistics/analytics, S-VIP monetization
- **Differentiators:** Partnership with Livo for measurement/valuation, claims "#1 platform in market"
- **Weaknesses:** Georgian-only, no mobile app, no AI, no reviews, no agent dashboard, basic design
- **SEO strength:** "N1 უძრავი ქონების პლატფორმა ბაზარზე" title claim
- **Monetization:** S-VIP badges, listing boosts

### 2. SS.ge — The Classifieds Giant

- **Type:** General classifieds (სიტყვა და საქმე), real estate is one vertical
- **Verticals:** Jobs, Services, Sales, Finance, Auto, Real Estate
- **Key features:** AI chatbot (chat.ss.ge) — FIRST to market with AI, Instagram-like "Stories" for listings, Cadastral map integration, Price calculator, SS Premium membership
- **Differentiators:** Cross-category traffic, AI assistant, Stories UX pattern for listings (unique in real estate)
- **Weaknesses:** Not real-estate-focused, less premium UX, emoji in UI, USD-first pricing, no agent tools
- **SEO strength:** General classifieds domain authority
- **Monetization:** SS Premium, promo services, VIP listings (SV badge with 494 count)

### 3. Korter.ge — The Data King

- **Established:** 2018
- **International:** korter.com (worldwide ambition)
- **Key features:** Native iOS + Android apps, WhatsApp notification subscriptions (unique moat), massive listing inventory (30K+ sale, 68K+ rent in Tbilisi), developer/construction company ecosystem, educational guides ("სახლის ყიდვის წესები"), HelpCrunch live chat
- **Differentiators:** WhatsApp alerts (no competitor has this), native mobile apps, largest inventory, developer ecosystem depth
- **Weaknesses:** Georgian-only, no AI, no reviews, no agent dashboard, no dark mode
- **SEO strength:** City-level pages with listing counts (e.g., "ბინების ყიდვა გაყიდვა ბათუმში 32 934"), mega-menu structure
- **Monetization:** Listing fees, agent subscriptions

### 4. Livo.ge — The Service-Rich Platform

- **Owner:** TNET group (sister to MyHome.ge)
- **Key features:** Livo AI (AI search), Real estate auctions (auction.livo.ge), Professional services: 3D tours, photography, measurement, valuation, Property transfer/registration (via tnet.ge), Premier Agent program, Facebook ads service for listings, Metro station search, Blog
- **Differentiators:** Most full-service platform, auctions (unique in Georgia), 3D virtual tours, professional photography service
- **Weaknesses:** Georgian-only, no mobile app, no reviews, no agent dashboard, design identical to MyHome (TNET template)
- **SEO strength:** District-level pages, sitemap, blog content
- **Monetization:** Premier Agent subscriptions, S-VIP badges, service fees (3D, photo, valuation)

## Critical Gaps — What We Must Ship

### Tier 1 (Score 10/10) — Ship This Week

1. **AI Search** — SS.ge (chat.ss.ge) and Livo (Livo AI) both have it. We have Gemini planned and AISection component shell. Wire it. This is the #1 competitive gap.

2. **Price/m² Everywhere** — All 4 competitors show price per square meter prominently. We show it only in image overlay for sale listings. Add to card body, search results, detail page.

3. **WhatsApp + Email Alerts** — Korter's WhatsApp alerts are their unique moat. No competitor has email alerts. Ship both: "Get new listings on WhatsApp" and "Email me when similar properties appear."

### Tier 2 (Score 9/10) — This Sprint

4. **Real-Time Search** — Our `filterListings()` on static arrays is MVP-debt. Wire Meilisearch for instant search with typo tolerance.

5. **Metro Station Proximity** — Livo's "იყიდება მეტროსთან" is unique and relevant for Tbilisi. Add metro station data to listings.

6. **Mortgage Calculator** — MyHome owns this keyword. We have brand slider CSS in globals.css. Build the calculator component.

### Tier 3 (Score 8/10) — Next Sprint

7. **Agent Subscription Program** — SS Premium + Livo Premier Agent monetize the supply side. Build agent tiers with listing boost, featured placement, analytics.

8. **Property Comparison** — No Georgian competitor has this. Add compare checkboxes to listing cards, comparison table view.

9. **AI Price Estimation** — Nobody has this yet. First-mover advantage. Use Gemini to estimate property value based on location, area, market data.

### Tier 4 (Score 7/10) — This Quarter

10. **Mobile App Deployment** via Capacitor
11. **3D Virtual Tours** — Livo offers as paid service. We can do better with Matterport/Cloudflare Stream integration.
12. **Automated Competitor Monitoring** — Dashboard that tracks competitor listing counts, new features, price trends.

## Strategic Recommendations

### Differentiate, Don't Copy

1. **AI-first platform** — Beat SS.ge and Livo by having the best AI, not just any AI. Gemini-powered search, price estimation, neighborhood insights.
2. **International buyers** — Our 9-language i18n is a moat. Target Russian, Turkish, Arabic, Israeli buyers who competitors can't serve.
3. **Supply-side tools** — Agent/agency/developer dashboards are unique. Double down. Make sivrce the platform agents need.
4. **Trust & transparency** — Reviews, verified listings, AI scores. No competitor has this.
5. **Speed** — PWA with offline support. Korter has apps but they're slow to update.
